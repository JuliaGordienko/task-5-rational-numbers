## Проект task 5: rational numbers
Требуется Python 3.12 или выше.

### Установка и запуск
Установка (Linux / macOS)
python3 -m venv .venv
source .venv/bin/activate\

### Установка (Windows Powershell)
python -m venv .venv
.venv\Scripts\Activate.ps1\

### Зaпycк
python main.py

Задание
Рациональные числа являются счетным множеством.
Реализовать генератор (очевидно, неограниченный), выдающий все рациональные числа по одному разу.
